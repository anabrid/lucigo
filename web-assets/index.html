LUCIGO Webserver

This file is served by LUCIGO, the administrative client for the digital-analog computer LUCIDAC written by anabrid.

Machine readable information about the hosting situation is available at /.well-known/lucidac.json. Typically, there is a websocket endpoint at /ws and the LUCIDAC GUI files are available at lucigui. This is also where you want to head to.

However, if you see this page it typically means that the Lucigui is not embedded. You can, however, head over to lucidac.online and enjoy the GUI there.
